# 🌟 GlowNest - Backend

Backend API for the GlowNest E-Commerce application built with Node.js, Express.js, MongoDB, and JWT Authentication.

## 🚀 Live API

https://final-project-backend-uvpl.onrender.com

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Multer
- Razorpay
- Bcrypt.js
- CORS

## ✨ Features

- User Authentication
- JWT Authorization
- Product APIs
- Wishlist APIs
- Cart APIs
- Profile APIs
- Order APIs
- Review APIs
- Razorpay Payment Integration
- Image Upload

## 📁 Installation

```bash
git clone <backend-repository-url>
cd backend
npm install
npm start
```

## ⚙️ Environment Variables

Create a `.env` file.

```env
PORT=5000

MONGO_URI=your_mongodb_connection

JWT_SECRET=your_jwt_secret

RAZORPAY_KEY_ID=your_key

RAZORPAY_KEY_SECRET=your_secret
```
